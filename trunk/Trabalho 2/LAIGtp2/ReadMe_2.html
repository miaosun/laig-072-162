TinyXML

Utilização do exemplo junto

  • A bilbioteca (TinyXML.lib) encontra-se incluída no projecto e é referida no código,
  • Inclui-se o header tinyxml.h (#include "tinyxml.h")
  • No main, invoca-se LoadScene()
  • Em SceneLoader, lê-se o XML para estruturas de dados baseadas na classe "map".
  • A rotina "display" invoca "desenhaPolygonos()" que visita as estruturas de dados anteriores e desenha, em OpenGL, os polígonos declarados no ficheiro XML.

O XML encontra-se no directório do projecto.

NOTA: este XML não tem qualquer semelhança com o que será necessário tratar no segundo trabalho.
